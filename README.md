<h2>🏠 HOME</h2>

<p>
Welcome to the <b>ATM Simulator Project</b>.  
This is a Python-based GUI banking system made using Tkinter.  
It simulates real ATM features like login, deposit, withdraw, transfer, and balance checking.
</p>

<p>
👉 This project is designed for learning Python GUI development and real-world logic building.
</p>

<h2>⚙️ How It Works</h2>

<ol>
  <li>🔐 User enters PIN to access system</li>
  <li>🏠 Home/Menu screen appears</li>
  <li>💰 Choose actions:
    <ul>
      <li>Deposit money</li>
      <li>Withdraw money</li>
      <li>Transfer money</li>
      <li>Check balance</li>
    </ul>
  </li>
  <li>📊 Balance updates instantly after every transaction</li>
</ol>

---

<h2>🚀 Features</h2>

<ul>
